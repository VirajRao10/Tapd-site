Help and troubleshooting

Support

Tapd needs Anki Desktop, the Tapd Anki add-on, your iPhone, and your Apple Watch on the same local Wi-Fi network.

Setup Checklist

  1. Install the Tapd add-on from AnkiWeb in Anki Desktop and restart Anki.
  2. Open Anki and choose Tools -> Tapd Watch Controller.
  3. Start a review session in Anki.
  4. Open Tapd on iPhone and enter the IP address, port, and 6-digit pairing code shown in Anki.
  5. Tap Send to Watch, then open Tapd on Apple Watch to complete pairing.
  6. Choose a controller mode and start reviewing.

Troubleshooting

  • Confirm Anki, iPhone, and Apple Watch are on the same Wi-Fi network.
  • Confirm the Watch is paired to the same iPhone where Tapd is installed.
  • Use the local IP address shown in the Tapd Watch Controller window, not a public IP address.
  • You can paste either the IP address or the full connection URL shown by the add-on into the iPhone app.
  • If the iPhone says Open Watch app, launch Tapd on Apple Watch and tap Send to Watch again.
  • If needed, the Watch pairing screen can still accept the IP address, port, and code directly.
  • Regenerate the pairing code if more than 10 minutes have passed.
  • Check that macOS Firewall or network security tools allow Anki to receive local connections.
  • If macOS asks whether Anki can accept incoming network connections, choose Allow.
  • Keep Anki open and in a review session before sending review commands.

Support Contact

For setup questions, pairing issues, and bug reports, open a support issue on the public Tapd website tracker: github.com/VirajRao10/Tapd-site/issues. If you connect a custom domain later, keep this support page available so users have a stable help URL.
