How to use it

Pair Anki and your Watch

  1. 1

    Install the Anki add-on

    Install the Tapd add-on from AnkiWeb, then restart Anki Desktop.

  2. 2

    Open the controller window

    In Anki, choose Tools -> Tapd Watch Controller. The add-on shows your local IP, port, and a 6-digit pairing code.

  3. 3

    Pair from Apple Watch

    Open Tapd on the Watch, enter the IP address, port, pairing code, and device name, then tap Pair.

  4. 4

    Start reviewing

    Open a review session in Anki, choose a Tapd mode on the Watch, and control the reviewer from your wrist.

Controller modes

Choose the control style that fits the session

Button Mode

Full controls for Show Answer, Again, Hard, Good, Easy, undo, bury, and suspend.

Minimal Mode

Tap for the primary action, long press for Again, and swipe for Hard or Easy.

Crown Mode

Rotate the Digital Crown to pick an answer ease, then tap to submit.

Double Tap Mode

Uses the watchOS primary hand gesture for show-answer and Good, with fallback buttons.

Gesture Mode

Shake, flick, or twist your wrist with sensitivity and cooldown controls.

Install

Get Tapd on Apple Watch and AnkiWeb

Tapd requires the watchOS app and the Tapd Anki add-on. Install the Watch app from the App Store, then install the Tapd add-on from AnkiWeb so Anki Desktop can accept local commands from your Watch.
